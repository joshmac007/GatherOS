# Open Source Acknowledgments

GatherOS is built on top of a stack of generous open source software. Many thanks to all the maintainers and contributors of these projects.

## Runtime dependencies

- **[better-sqlite3](https://github.com/WiseLibs/better-sqlite3)** — MIT License
  Embedded SQLite engine. Backs the local library database.

- **[electron-updater](https://github.com/electron-userland/electron-builder)** — MIT License
  Auto-update channel against GitHub Releases.

- **[nanoid](https://github.com/ai/nanoid)** — MIT License
  ID generation for saves, collections, and tags.

- **[node-vibrant](https://github.com/Vibrant-Colors/node-vibrant)** — MIT License
  Color palette extraction from saved images.

- **[sharp](https://github.com/lovell/sharp)** — Apache License 2.0
  Image decoding, resizing, and thumbnail generation.

## Transitive dependencies

The packages listed above pull in further dependencies under permissive licenses (mostly MIT, ISC, BSD, and Apache-2.0). Each retains its original copyright and license text inside `node_modules/<package>/`.
